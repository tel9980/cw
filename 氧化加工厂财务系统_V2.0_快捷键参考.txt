#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
快捷键参考卡片

使用快捷键可以大幅提升工作效率！

═══════════════════════════════════════════════════════════════════════
                           通用快捷键
═══════════════════════════════════════════════════════════════════════

  F1      │ 显示帮助
  F5      │ 刷新数据
  Ctrl+S  │ 保存当前操作
  Ctrl+N  │ 新建（订单/客户/供应商）
  Ctrl+Z  │ 撤销
  Ctrl+Y  │ 重做
  Ctrl+C  │ 复制
  Ctrl+V  │ 粘贴
  Ctrl+F  │ 搜索
  Esc     │ 取消/关闭

═══════════════════════════════════════════════════════════════════════
                          订单管理
═══════════════════════════════════════════════════════════════════════

  Ctrl+O │ 新建订单
  F2      │ 编辑选中订单
  Delete  │ 删除选中订单
  Enter   │ 查看订单详情
  Alt+C   │ 复制订单
  Alt+P   │ 打印订单

  状态筛选:
    1      │ 待加工
    2      │ 加工中
    3      │ 已完工
    4      │ 已交付
    5      │ 已收款

═══════════════════════════════════════════════════════════════════════
                          收支管理
═══════════════════════════════════════════════════════════════════════

  Ctrl+I │ 新建收入
  Ctrl+E │ 新建支出
  Alt+A  │ 分配收入到订单
  Alt+S  │ 分配支出到订单

  快速输入金额:
    @     │ 输入今日收入合计
    #     │ 输入今日支出合计
    $     │ 显示本月汇总

═══════════════════════════════════════════════════════════════════════
                          客户/供应商
═══════════════════════════════════════════════════════════════════════

  Alt+C  │ 新建客户
  Alt+S  │ 新建供应商
  F3     │ 查找客户
  F4     │ 查找供应商
  Tab    │ 切换到下一个字段
  Shift+Tab │ 切换到上一个字段

═══════════════════════════════════════════════════════════════════════
                          报表功能
═══════════════════════════════════════════════════════════════════════

  Ctrl+R │ 生成利润表
  Ctrl+T │ 生成成本报表
  Ctrl+K │ 生成客户对账单
  Ctrl+L │ 生成供应商对账单
  Alt+E  │ 导出报表到Excel
  Alt+P  │ 打印报表

═══════════════════════════════════════════════════════════════════════
                          数据工具
═══════════════════════════════════════════════════════════════════════

  Ctrl+B │ 备份数据
  Ctrl+D │ 数据导入
  Alt+I  │ 数据导出
  Ctrl+U │ 工具菜单

═══════════════════════════════════════════════════════════════════════
                          窗口操作
═══════════════════════════════════════════════════════════════════════

  Ctrl+Tab   │ 切换标签页
  Ctrl+W     │ 关闭当前标签
  Ctrl+T     │ 新建标签页
  Alt+1~9    │ 跳转到标签页1~9
  Ctrl++     │ 放大字体
  Ctrl+-     │ 缩小字体
  Ctrl+0     │ 恢复默认字体

═══════════════════════════════════════════════════════════════════════
                          快捷命令（命令行）
═══════════════════════════════════════════════════════════════════════

  >stats   │ 显示今日统计
  >week    │ 显示本周统计
  >month   │ 显示本月统计
  >orders  │ 显示待处理订单
  >search  │ 搜索（>search 张三）
  >top     │ TOP客户排行
  >exit    │ 退出

═══════════════════════════════════════════════════════════════════════
                          移动端快捷键
═══════════════════════════════════════════════════════════════════════

  ← →   │ 左右滑动切换页面
  ↑ ↓   │ 上下滑动列表
  +     │ 新建
  ≡     │ 菜单
  ↩     │ 返回

═══════════════════════════════════════════════════════════════════════
                          效率技巧
═══════════════════════════════════════════════════════════════════════

  1. 批量操作
     • 勾选多个订单 → 右键批量操作
     • 批量导出: Ctrl+鼠标选择 → 右键导出

  2. 快速输入
     • 输入金额后按 Tab 自动计算
     • 输入客户名后按 Enter 快速搜索

  3. 常用报表
     • 报表可保存为模板
     • 快捷调用: Alt+R + 模板名

  4. 数据导入
     • Excel导入: 准备模板 → Ctrl+D
     • 批量导入: 选中文件 → 拖拽

═══════════════════════════════════════════════════════════════════════
                           提示
═══════════════════════════════════════════════════════════════════════

  • 忘记快捷键？按 F1 查看帮助
  • 自定义快捷键: 设置 → 快捷键
  • 禁用快捷键: 设置 → 快捷键 → 禁用
