# V1.7 氧化加工厂智能财务助手完整版

## 项目概述

V1.7是专为小型氧化加工厂设计的全功能智能财务管理系统,整合了:

- **V1.6实用增强版**: 核心财务功能(存储、导入、报表、对账、提醒)
- **V1.5工作流优化版**: 智能工作流(智能工作台、工作流引擎、一键操作)
- **氧化加工厂行业特色**: 多种计价单位、外发加工管理、行业费用分类

## 项目结构

```
oxidation_complete_v17/
├── models/              # 数据模型(V1.6 + V1.5 + 行业扩展)
├── storage/             # 数据存储(V1.6)
├── core/                # 核心功能(V1.6)
├── config/              # 配置管理(V1.6)
├── import_engine/       # Excel导入引擎(V1.6)
├── reports/             # 报表生成(V1.6 + 行业报表)
├── reconciliation/      # 对账功能(V1.6 + 灵活对账)
├── reminders/           # 提醒系统(V1.6)
├── ui/                  # 用户界面(V1.6 + 行业菜单)
├── workflow/            # 智能工作流(V1.5)
├── industry/            # 行业特色功能(新增)
└── tests/               # 测试套件
```

## 核心特性

### 氧化加工厂行业特色
- ✅ 多种计价单位支持(件、条、只、个、米长、米重、平方)
- ✅ 外发加工管理(喷砂、拉丝、抛光)
- ✅ 行业费用自动分类(原材料、挂具、外发加工费等)
- ✅ 灵活对账(一对多、多对一)
- ✅ 银行账户分类(有票据/现金账户)

### 智能工作流(V1.5)
- ✅ 智能工作台(今日优先任务)
- ✅ 工作流引擎(早晨准备、交易录入、日终汇总)
- ✅ 一键操作(常见交易快速录入)
- ✅ 智能学习和建议

### 实用核心功能(V1.6)
- ✅ Excel批量导入
- ✅ 智能报表生成
- ✅ 银行对账助手
- ✅ 智能提醒系统
- ✅ 往来单位管理
