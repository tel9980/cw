## 💡 核心业务逻辑

### 1. 订单利润计算
```python
订单利润 = 总加工费 - 委外成本 - 分摊的固定成本

例如：
订单OX202401001
- 总加工费：500元
- 委外成本：100元（喷砂）
- 分摊固定成本：50元（按订单金额比例分摊）
- 订单利润：350元
- 利润率：70%
```

### 2. 收款分配逻辑
```python
场景1：一笔收款对应一个订单
- 直接分配：收款金额 → 订单

场景2：一笔收款对应多个订单
- 按比例分配：收款金额 → 订单1 + 订单2 + ...
- 或手动指定每个订单的分配金额

场景3：预收款
- 先记录收款
- 订单完成后再分配
- 支持部分分配
```

### 3. 应收应付管理
```python
应收账款 = 订单总金额 - 已收金额
应付账款 = 采购总金额 - 已付金额

账龄分析：
- 30天内：正常
- 30-60天：提醒
- 60-90天：催收
- 90天以上：重点关注
```

### 1. 利润表
```

减：营业成本
  - 原料成本（三酸、片碱、亚钠等）
  - 委外加工成本
  - 人工成本（工资）
= 毛利润

减：期间费用
  - 房租
  - 水电费
  - 日常费用
= 净利润
```

```
资产：
  - 银行存款（G银行 + N银行）

  - 其他负债
```
