## 📊 日常需要维护的表格

### 核心表格（必须维护）⭐⭐⭐

#### 1. 交易记录表（流水账）

**文件名**: `我的流水账.xlsx`

**列名**（7列）:
| 日期 | 类型 | 金额 | 往来单位 | 摘要 | 类别 | 状态 |
|------|------|------|----------|------|------|------|
| 2026-02-10 | 收入 | 5000 | 张三公司 | 销售货款 | 产品销售 | 已完成 |
| 2026-02-10 | 支出 | 1200 | 李四供应商 | 采购原料 | 原材料采购 | 已完成 |

**维护频率**: 每天或每周
**时间成本**: 5-10分钟/天

**小技巧**:
- 收到钱/付钱时立即记录
- 可以用手机拍照，晚上再录入
- 摘要写清楚，方便以后查

#### 2. 往来单位表（客户供应商）

**文件名**: `客户供应商名单.xlsx`

**列名**（7列）:
| 单位名称 | 类型 | 联系人 | 电话 | 邮箱 | 地址 | 税号 |
|----------|------|--------|------|------|------|------|
| 张三公司 | 客户 | 张经理 | 138... | zhang@... | 北京... | 91... |
| 李四供应商 | 供应商 | 李总 | 139... | li@... | 上海... | 91... |

**维护频率**: 有新客户/供应商时添加
**时间成本**: 2分钟/个

**小技巧**:
- 第一次合作时就录入
- 保存名片信息
- 税号很重要（开票用）

### 辅助表格（可选维护）⭐⭐

#### 3. 银行流水表（对账用）

**文件名**: 从银行下载的Excel

**列名**（6列）:
| 交易日期 | 摘要 | 收入金额 | 支出金额 | 余额 | 对方户名 |
|----------|------|----------|----------|------|----------|
| 2026-02-10 | 转账 | 5000 | | 105000 | 张三公司 |

**维护频率**: 每月一次（月底）
**时间成本**: 5分钟（直接从网银下载）

**小技巧**:
- 登录网银 → 下载Excel
- 不需要手动录入
- 用于月底对账

## 🔄 日常工作流程

### 每天（5-10分钟）

```
早上/晚上：
1. 打开 我的流水账.xlsx
2. 记录今天的收入支出
3. 保存文件
```

**示例**:
```
今天收到张三公司货款5000元
→ 打开Excel
→ 新增一行：2026-02-10 | 收入 | 5000 | 张三公司 | 销售货款 | 产品销售 | 已完成
→ 保存
```

### 每周（可选）

```
周五下午：
1. 检查本周流水账是否都记录了
2. 补录遗漏的交易
```

### 每月（30分钟）

```
月初（1-5号）：
1. 从网银下载上月银行流水
2. 导入小会计系统
3. 双击"生成月度报表.bat"
4. 查看生成的报表
5. 备份数据
```

**详细步骤**:

#### 第1步：下载银行流水（5分钟）
```
登录网银 → 查询交易明细 → 选择上月 → 导出Excel
```

#### 第2步：导入数据（10分钟）
```
1. 双击 start.bat 启动系统
2. 选择"Excel批量导入"
3. 选择"导入交易记录" → 选择你的流水账.xlsx
4. 预览 → 确认 → 导入
5. 选择"导入银行流水" → 选择下载的银行流水.xlsx
6. 预览 → 确认 → 导入
```

#### 第3步：生成报表（1分钟）
```
双击 生成月度报表.bat
等待30秒
完成！
```

#### 第4步：查看报表（10分钟）
```
打开 reports/ 文件夹
查看4个报表：
- 管理报表.xlsx（老板自己看）
- 增值税申报表.xlsx（报税用）
- 所得税申报表.xlsx（报税用）
- 银行贷款报表.xlsx（贷款用）
```

#### 第5步：备份（2分钟）
```
复制 data/ 文件夹
粘贴到 U盘 或 网盘
```

### 每季度（1小时）

```
季度末：
1. 生成季度报表（3个月合并）
2. 打印重要报表存档
3. 检查应收应付账款
4. 发送催款函（系统自动生成）
```

### 每年（2小时）

```
年底：
1. 生成年度报表
2. 整理全年数据
3. 归档历史数据
4. 准备年度审计材料
```

## 💡 实用技巧

### 技巧1：快速记账

**方法1：手机拍照法**
```
收到发票/单据 → 手机拍照 → 晚上看照片录入Excel
```

**方法2：微信记录法**
```
发生交易 → 发微信给自己 → 晚上整理录入Excel
```

**方法3：日记本法**
```
随身带小本子 → 记录交易 → 晚上录入Excel
```

### 技巧2：分类管理

**收入类别**（常用）:
- 产品销售
- 服务收入
- 咨询费
- 技术服务费
- 租金收入

**支出类别**（常用）:
- 原材料采购
- 设备租赁
- 物流费用
- 办公费用
- 工资支出
- 水电费
- 房租

**自定义类别**:
- 根据自己行业添加
- 保持类别一致
- 不要太细分

### 技巧3：往来单位管理

**客户编号规则**:
```
CUST001 - 张三公司
CUST002 - 王五客户
CUST003 - 赵六公司
```

**供应商编号规则**:
```
SUPP001 - 李四供应商
SUPP002 - 办公用品店
SUPP003 - 物流公司
```

### 技巧4：月底对账

**对账清单**:
- [ ] 银行流水已下载
- [ ] 所有交易已录入
- [ ] 金额核对无误
- [ ] 往来单位信息完整
- [ ] 报表已生成
- [ ] 数据已备份

### 技巧5：数据备份

**备份频率**:
- 每月备份一次（必须）
- 重要交易后立即备份
- 生成报表后备份

**备份方法**:
```
方法1：U盘备份
复制 data/ 文件夹 → 粘贴到U盘

方法2：网盘备份
复制 data/ 文件夹 → 上传到百度网盘/阿里云盘

方法3：多地备份
同时备份到U盘 + 网盘 + 移动硬盘
```

## 🆘 常见问题

### Q1: 我不会编程，能用吗？
**A**: 能！完全不需要编程知识，只需要会用Excel。

### Q2: 我Excel也不太会，怎么办？
**A**: 没关系！只需要会：
- 打开Excel文件
- 在表格里输入文字和数字
- 保存文件
这些小学生都会！

### Q3: 每天必须记账吗？
**A**: 不必须，但建议：
- 最好每天记（5分钟）
- 至少每周记一次
- 月底前必须补全

### Q4: 忘记记账怎么办？
**A**: 
- 查看银行流水补录
- 查看微信/支付宝记录
- 查看发票/单据
- 实在想不起来就标注"其他"

### Q5: 数据会丢失吗？
**A**: 不会！只要：
- 定期备份（每月一次）
- 保存到多个地方
- 不要删除data文件夹
