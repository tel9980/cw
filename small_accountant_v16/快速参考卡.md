# 小会计V1.6 - 快速参考卡

## 🚀 5分钟快速上手

### 第一步：安装（只需一次）
```
双击 install.bat → 等待安装完成
```

### 第二步：准备数据
```
打开 Excel模板/01_交易记录模板.xlsx
删除示例数据，录入自己的数据
保存文件
```

### 第三步：导入数据
```
双击 导入Excel数据.bat → 选择文件 → 确认导入
```

### 第四步：生成报表
```
双击 生成月度报表.bat → 等待30秒 → 完成！
```

## 🆘 常见问题

### Q1: 导入Excel失败？
**A**: 检查Excel格式是否正确，使用提供的模板

### Q2: 报表生成失败？
**A**: 确保已导入交易数据，至少有1条记录

### Q3: 找不到生成的报表？
**A**: 报表在 `reports/` 文件夹中

### Q4: 如何修改错误数据？
**A**: 在 `data/` 文件夹中找到对应的JSON文件，用记事本打开修改

### Q5: 如何备份数据？
**A**: 双击"数据备份.bat"，或手动复制 `data/` 文件夹

### Q6: 如何恢复数据？
**A**: 从备份文件夹中复制 `data/` 文件夹回来

### Q7: 可以多台电脑使用吗？
**A**: 可以，复制整个文件夹到U盘，在其他电脑上运行

### Q8: 需要联网吗？
**A**: 不需要，完全本地运行

### Q9: 数据安全吗？
**A**: 数据存储在本地，建议定期备份

### Q10: 可以长期使用吗？
**A**: 可以，适合3-5年长期使用
