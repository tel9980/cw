**旁白**:
```

删除示例数据，保留标题行
然后录入你自己的数据
```

## 📹 教程4：常见问题解决（故障排除）

**时长**: 6分钟  
**目标**: 教会用户解决常见问题

### 问题1：导入失败（1分钟）

**画面**: 演示导入失败的解决方法

**旁白**:
```
导入Excel失败怎么办？

常见原因：
1. Excel格式不对
   - 解决：使用提供的模板
   - 不要修改标题行

2. 数据格式错误
   - 日期格式：YYYY-MM-DD
   - 金额格式：数字，不要有逗号
   - 类型：从下拉菜单选择

3. 必填字段为空
   - 日期、类型、金额、往来单位必填
   - 备注可以为空

4. 文件被占用
   - 关闭Excel文件
   - 再次导入

如果还是失败，查看错误消息
根据提示修改数据
```

---

### 问题2：报表生成失败（1分钟）

**画面**: 演示报表生成失败的解决方法

**旁白**:
```
报表生成失败怎么办？

常见原因：
1. 没有数据
   - 至少导入1条交易记录
   - 检查数据是否导入成功

2. 日期范围错误
   - 确保有指定月份的数据
   - 检查交易记录的日期

3. 文件被占用
   - 关闭之前生成的报表
   - 再次生成

4. 磁盘空间不足
   - 清理磁盘空间
   - 删除旧的报表

如果还是失败，查看错误日志
在 logs 文件夹中
```

---
