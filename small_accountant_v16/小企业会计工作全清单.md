# 小企业会计工作全清单

## 📋 概述

小企业会计不只是记流水账，还要处理各种财务相关工作。本清单涵盖小企业会计的**所有日常工作**，帮助你全面掌握财务管理。

---

## 🗓️ 每日工作（30-60分钟）

### 1. 记录交易流水 ⭐⭐⭐
**时间**: 10-15分钟  
**工作内容**:
- 记录当天所有收入（销售、服务费等）
- 记录当天所有支出（采购、费用等）
- 记录银行转账、现金收付
- 记录订单、合同签订

**使用工具**: Excel流水账表 + 小会计系统

**小技巧**:
```
收到钱 → 立即记录 → 保存发票/单据
付出钱 → 立即记录 → 保存付款凭证
```

### 2. 整理单据票据 ⭐⭐⭐
**时间**: 10-15分钟  
**工作内容**:
- 收集当天的发票（销项、进项）
- 整理收据、付款凭证
- 扫描或拍照重要单据
- 按日期分类存档

**存档方法**:
```
按月份建文件夹：
  2026年02月/
    ├── 01日_发票/
    ├── 02日_发票/
    └── ...
```

### 3. 核对现金库存 ⭐⭐
**时间**: 5-10分钟  
**工作内容**:
- 盘点现金余额
- 核对与账面金额是否一致
- 记录现金日记账

**注意事项**:
- 现金和账面必须一致
- 有差异立即查找原因
- 每天下班前核对

### 4. 查看银行余额 ⭐⭐
**时间**: 5分钟  
**工作内容**:
- 登录网银查看余额
- 核对大额收支是否到账
- 关注账户异常情况

### 5. 处理紧急事项 ⭐
**时间**: 10-20分钟  
**工作内容**:
- 开具发票（客户要求）
- 处理报销申请
- 回复财务咨询
- 处理付款审批

## 📅 每周工作（1-2小时）

### 1. 整理本周账目 ⭐⭐⭐
**时间**: 30-45分钟  
**工作内容**:
- 检查本周流水账是否完整
- 补录遗漏的交易
- 核对发票与流水账
- 整理单据归档

### 2. 应收账款跟进 ⭐⭐⭐
**时间**: 15-30分钟  
**工作内容**:
- 查看哪些客户欠款
- 计算账龄（欠款多少天）
- 联系即将到期的客户
- 发送对账单或催款提醒

**使用小会计**:
```
系统自动提醒 → 查看逾期客户 → 生成催款函 → 发送
```

### 3. 应付账款管理 ⭐⭐⭐
**时间**: 15-30分钟  
**工作内容**:
- 查看哪些供应商要付款
- 确认付款日期和金额
- 准备付款申请
- 安排资金支付

### 4. 发票管理 ⭐⭐
**时间**: 15-20分钟  
**工作内容**:
- 开具本周销项发票
- 收集本周进项发票
- 检查发票真伪（税务局网站）
- 整理发票清单

### 5. 资金计划 ⭐⭐
**时间**: 10-15分钟  
**工作内容**:
- 预测下周资金需求
- 检查账户余额是否充足
- 安排大额支付计划
- 关注现金流预警

## 📆 每月工作（4-8小时）

### 1. 月度对账 ⭐⭐⭐
**时间**: 1-2小时  
**工作内容**:
- 下载银行流水对账
- 核对每笔交易是否记录
- 查找未达账项（已记账但未到账）
- 调整差异

**使用小会计**:
```
导入银行流水 → 自动匹配 → 查看未匹配项 → 手动处理
```

### 2. 生成月度报表 ⭐⭐⭐
**时间**: 30-60分钟  
**工作内容**:
- 生成收支明细表
- 生成利润表（收入-支出）
- 生成资产负债表
- 生成现金流量表

**使用小会计**:
```
双击"生成月度报表.bat" → 等待30秒 → 完成！
```

### 3. 税务申报准备 ⭐⭐⭐
**时间**: 1-2小时  
**工作内容**:
- 整理增值税发票（销项、进项）
- 计算应交增值税
- 准备所得税申报数据
- 填写税务申报表

**申报时间**:
- 增值税：每月15日前
- 所得税：每季度15日前（按季申报）

**使用小会计**:
```
生成增值税申报表 → 生成所得税申报表 → 登录税务局网站 → 填报
```

### 4. 工资核算 ⭐⭐⭐
**时间**: 1-2小时  
**工作内容**:
- 统计员工考勤
- 计算工资（基本工资+绩效+补贴）
- 计算社保、公积金
- 计算个人所得税
- 生成工资表
- 发放工资

**工资表示例**:
```
姓名 | 基本工资 | 绩效 | 补贴 | 应发 | 社保 | 个税 | 实发
张三 | 5000 | 1000 | 500 | 6500 | 800 | 150 | 5550
```

### 5. 往来对账 ⭐⭐
**时间**: 1-2小时  
**工作内容**:
- 与主要客户对账（应收账款）
- 与主要供应商对账（应付账款）
- 生成对账单发送确认
- 处理对账差异

**使用小会计**:
```
选择客户 → 生成对账单 → 导出Excel → 发送邮件
```

### 6. 发票认证抵扣 ⭐⭐
**时间**: 30-60分钟  
**工作内容**:
- 登录增值税发票平台
- 勾选本月进项发票
- 确认抵扣
- 下载认证清单

**注意事项**:
- 发票必须在开票日起360天内认证
- 每月申报前完成认证

### 7. 固定资产折旧 ⭐
**时间**: 15-30分钟  
**工作内容**:
- 计算本月固定资产折旧
- 更新固定资产台账
- 记录折旧费用

**折旧计算**:
```
月折旧额 = 资产原值 × 月折旧率
例：电脑5000元，5年折旧
月折旧 = 5000 ÷ 60个月 = 83.33元
```

### 8. 费用报销审核 ⭐⭐
**时间**: 30-60分钟  
**工作内容**:
- 审核员工报销单
- 检查发票真实性
- 核对报销金额
- 安排付款

### 9. 数据备份 ⭐⭐⭐
**时间**: 10-15分钟  
**工作内容**:
- 备份财务数据
- 备份Excel表格
- 备份发票扫描件
- 保存到多个地方（U盘+网盘）

## 📊 每季度工作（8-12小时）

### 1. 季度所得税申报 ⭐⭐⭐
**时间**: 2-3小时  
**工作内容**:
- 汇总季度收入
- 汇总季度成本费用
- 计算季度利润
- 计算应交所得税
- 填报所得税申报表

**申报时间**: 每季度结束后15日内

### 2. 季度财务分析 ⭐⭐
**时间**: 2-3小时  
**工作内容**:
- 分析季度收入变化
- 分析季度成本费用
- 分析利润率
- 分析现金流
- 编写财务分析报告

### 3. 应收账款清理 ⭐⭐⭐
**时间**: 2-3小时  
**工作内容**:
- 清理长期挂账的应收款
- 催收逾期账款
- 评估坏账风险
- 计提坏账准备

### 4. 库存盘点 ⭐⭐
**时间**: 2-3小时  
**工作内容**:
- 盘点实物库存
- 核对账面库存
- 处理盘盈盘亏
- 更新库存台账

---

## 📅 每年工作（20-40小时）

### 1. 年度汇算清缴 ⭐⭐⭐
**时间**: 8-12小时  
**工作内容**:
- 汇总全年收入
- 汇总全年成本费用
- 调整纳税差异
- 计算全年应交所得税
- 填报年度所得税申报表
- 提交税务局审核

**申报时间**: 次年5月31日前

### 2. 年度财务报表 ⭐⭐⭐
**时间**: 4-6小时  
**工作内容**:
- 编制年度资产负债表
- 编制年度利润表
- 编制年度现金流量表
- 编制财务报表附注

### 3. 年度审计准备 ⭐⭐
**时间**: 4-6小时  
**工作内容**:
- 整理全年账目
- 准备审计资料
- 配合审计师工作
- 处理审计发现的问题

### 4. 年度预算编制 ⭐⭐
**时间**: 4-6小时  
**工作内容**:
- 分析今年财务数据
- 预测明年收入
- 预算明年支出
- 编制年度预算表

### 5. 档案整理归档 ⭐⭐
**时间**: 4-6小时  
**工作内容**:
- 整理全年凭证
- 装订凭证册
- 整理发票档案
- 归档财务报表
- 移交档案室保管

## 📋 工作优先级

### 必须做（不能拖延）⭐⭐⭐
1. 记录交易流水（每天）
2. 税务申报（按时申报，避免罚款）
3. 工资发放（按时发放，员工满意）
4. 应付账款支付（维护供应商关系）
5. 发票管理（及时开票、认证）

### 重要做（尽快完成）⭐⭐
1. 应收账款催收（保证现金流）
2. 月度对账（及时发现问题）
3. 财务报表（了解经营状况）
4. 单据整理（方便查找）
5. 数据备份（防止丢失）

### 可以缓（有空再做）⭐
1. 财务分析（深入了解）
2. 制度建设（规范管理）
3. 档案整理（年底集中处理）

### 1. 养成好习惯
- 每天记账，不要拖延

- 按时申报，避免罚款

- 用Excel记流水账（简单高效）

- 税务合规（避免风险）
- 应收账款催收（保证回款）

- 学习税务政策（经常变化）
- 学习财务知识（提升专业）
- 学习软件工具（提高效率）
