# V1.6 小会计实用增强版 - 快速开始指南

## 欢迎使用！

V1.6小会计实用增强版是一个专为小微企业设计的财务管理系统，帮助您轻松完成日常财务工作。

## 5分钟快速上手

### 第一步：安装系统

```bash
# 1. 安装Python依赖
pip install -r requirements.txt

# 2. 验证安装
python -m pytest small_accountant_v16/tests/ -v
```

### 第二步：初始化配置

```bash
# 运行CLI启动系统
python small_accountant_v16/run_cli.py
```

首次运行会自动创建配置文件和数据目录。

### 第三步：开始使用

系统启动后，您会看到主菜单：

```
========================================
    V1.6 小会计实用增强版
========================================

请选择功能：
1. 📊 智能报表生成
2. 🔔 智能提醒管理
3. 🔄 快速对账助手
4. 📥 Excel批量导入
5. ⚙️  系统配置
0. 退出系统
```

## 三大核心功能

### 1️⃣ 智能报表生成（功能1）

**一键生成专业报表**

- **管理报表**：收支对比、利润趋势、客户排名
- **税务报表**：增值税、所得税申报表
- **银行贷款报表**：资产负债表、利润表、现金流量表

**使用步骤**：
1. 选择菜单 `1 - 智能报表生成`
2. 选择报表类型
3. 输入日期范围
4. 系统自动生成Excel报表

**示例**：
```
选择报表类型：
1. 管理报表
2. 税务报表
3. 银行贷款报表

请输入选择：1
开始日期（YYYY-MM-DD）：2026-01-01
结束日期（YYYY-MM-DD）：2026-12-31

✅ 报表生成成功！
文件位置：reports/管理报表_2026-01-01至2026-12-31.xlsx
```

### 2️⃣ 智能提醒管理（功能2）

**永不错过重要事项**

- **税务提醒**：增值税、所得税申报截止日期
- **应付账款提醒**：供应商付款到期提醒
- **应收账款提醒**：客户逾期账款催收
- **现金流预警**：资金短缺提前预警

**使用步骤**：
1. 选择菜单 `2 - 智能提醒管理`
2. 查看当前提醒或设置新提醒
3. 配置通知渠道（桌面/企业微信）

**提醒示例**：
```
📋 当前提醒列表：

🔴 高优先级
- 增值税申报截止（还有3天）
- 客户A逾期账款30天（金额：50,000元）

🟡 中优先级
- 供应商B付款到期（还有7天）
```

### 3️⃣ 快速对账助手（功能3）

**轻松完成银行对账**

- **银行流水对账**：自动匹配交易记录
- **客户对账单**：一键生成客户对账单
- **供应商对账**：核对应付账款

**使用步骤**：
1. 选择菜单 `3 - 快速对账助手`
2. 选择对账类型
3. 上传银行流水或选择往来单位
4. 系统自动匹配并生成对账报告

**对账示例**：
```
对账结果：
✅ 已匹配：45笔
⚠️  差异：3笔
❌ 未匹配：2笔

差异明细已导出到：reports/对账差异报告.xlsx
```

### 4️⃣ Excel批量导入（功能4）

**快速导入历史数据**

- **智能列识别**：自动识别Excel列名
- **数据验证**：导入前验证数据完整性
- **导入预览**：查看导入结果再确认
- **撤销功能**：支持撤销错误导入

**使用步骤**：
1. 准备Excel文件（参考模板）
2. 选择菜单 `4 - Excel批量导入`
3. 选择导入类型（交易记录/往来单位）
4. 选择Excel文件
5. 预览导入结果
6. 确认导入

**导入示例**：
```
📥 导入预览：

文件：交易记录_2026.xlsx
总行数：100
有效数据：98行
错误数据：2行

错误详情：
- 第15行：金额格式错误
- 第67行：日期格式错误

是否继续导入？(y/n)
```
