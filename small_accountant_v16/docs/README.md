## 💡 使用建议

### 推荐工作流程

1. **初始化设置**（首次使用）
   - 安装系统和依赖
   - 配置基本信息
   - 导入历史数据

2. **日常使用**
   - 定期导入交易记录
   - 查看智能提醒
   - 生成定期报表

3. **月末工作**
   - 银行流水对账
   - 生成客户对账单
   - 生成税务报表

4. **定期维护**
   - 备份数据
   - 清理历史记录
   - 更新系统配置

### 最佳实践

✅ **推荐做法**：
- 每天导入当天交易记录
- 每周查看提醒和预警
- 每月进行银行对账
- 定期备份数据
- 使用Excel模板导入

❌ **避免做法**：
- 手动编辑数据文件
- 长时间不备份数据
- 忽略系统提醒
- 一次导入过多数据
- 不验证导入结果

## 🔧 技术信息

### 系统要求
- Python 3.8+
- 100MB 磁盘空间
- 2GB 内存（推荐）

### 主要依赖
- openpyxl - Excel文件处理
- pandas - 数据分析
- matplotlib - 图表生成
- pytest - 测试框架

### 数据存储
- 格式：JSON
- 位置：`data/` 目录
- 备份：自动备份到 `backups/`

### 报表输出
- 格式：Excel (.xlsx)
- 位置：`reports/` 目录
- 包含：数据表格和可视化图表
