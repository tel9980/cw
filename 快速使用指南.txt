    氧化加工厂财务助手 - 快速使用指南
═══════════════════════════════════════════════════════════════════

📌 第一步：启动程序
─────────────────────────────────────────────────────────────────
  方式1：双击运行 "启动_氧化加工厂版.bat"
  方式2：在命令行运行 "python 氧化加工厂财务助手.py"

📌 第二步：生成示例数据（首次使用）
─────────────────────────────────────────────────────────────────
  1. 启动程序后，选择功能 "03 - 生成示例数据"
  2. 系统会自动生成5个示例文件：
     ✓ 示例_往来单位.xlsx（15个往来单位）
     ✓ 示例_加工订单.xlsx（10个订单）
     ✓ 示例_G银行流水.xlsx（有票流水）
     ✓ 示例_N银行微信流水.xlsx（现金流水）
     ✓ 氧化加工厂完整示例数据.xlsx（综合文件）
  3. 文件位置：财务数据/示例数据/

📌 第三步：学习订单录入
─────────────────────────────────────────────────────────────────
  1. 选择功能 "01 - 新建加工订单"
  2. 按照4步向导操作：
     步骤1：输入基本信息（订单编号、客户、日期、物品）
     步骤2：输入计价信息（单位、数量、单价）
     步骤3：输入工序信息（工序明细、外发工序、成本）
     步骤4：确认信息并创建
  3. 系统会自动计算订单金额和预计利润

📌 第四步：查看系统配置
─────────────────────────────────────────────────────────────────
  选择功能 "10 - 查看系统配置"
  可以查看：
  ✓ 7种计价单位（件/条/只/个/米长/米重/平方）
  ✓ 3种外发工序（喷砂/拉丝/抛光）
  ✓ 6种原材料类型
  ✓ 3个银行账户配置
  ✓ 完整的费用分类体系

📌 第五步：查看使用教程
─────────────────────────────────────────────────────────────────
  选择功能 "11 - 使用教程"
  包含：
  ✓ 快速开始指南
  ✓ 订单录入流程详解
  ✓ 计价单位说明
  ✓ 外发工序说明
  ✓ 详细文档链接

═══════════════════════════════════════════════════════════════════
    7种计价单位说明
═══════════════════════════════════════════════════════════════════

  件 - 按件数计价
       示例：1000件 × 2.5元/件 = 2500元

  条 - 按条数计价
       示例：300条 × 12元/条 = 3600元

  只 - 按只数计价
       示例：5000只 × 0.8元/只 = 4000元

  个 - 按个数计价
       示例：10000个 × 0.5元/个 = 5000元

  米长 - 按长度计价
       示例：500米 × 8元/米 = 4000元

  米重 - 按重量计价（公斤）
       示例：200公斤 × 15元/公斤 = 3000元

  平方 - 按面积计价
       示例：50平方 × 80元/平方 = 4000元

═══════════════════════════════════════════════════════════════════
    3种外发工序说明
═══════════════════════════════════════════════════════════════════

  喷砂 - 表面喷砂处理
         用于去除表面氧化层、毛刺等

  拉丝 - 表面拉丝处理
         用于形成均匀的拉丝纹理

  抛光 - 表面抛光处理
         用于提高表面光洁度

  💡 提示：外发工序可以多选，系统会自动计算预计利润

═══════════════════════════════════════════════════════════════════
    典型业务场景示例
═══════════════════════════════════════════════════════════════════

场景1：简单氧化订单
  客户：华强五金
  物品：铝合金把手
  计价：1000件 × 2.5元/件 = 2500元
  工序：氧化
  外发：无
  利润：2500元

场景2：含外发工序订单
  客户：精工制造
  物品：不锈钢管
  计价：300条 × 12元/条 = 3600元
  工序：喷砂→拉丝→氧化
  外发：喷砂（600元）+ 拉丝（400元）= 1000元
  利润：3600 - 1000 = 2600元

场景3：按重量计价订单
  客户：建筑五金
  物品：铝型材
  计价：200公斤 × 15元/公斤 = 3000元
  工序：氧化
  外发：无
  利润：3000元

场景4：按面积计价订单
  客户：装饰材料
  物品：铝板
  计价：50平方 × 80元/平方 = 4000元
  工序：抛光→氧化
  外发：抛光（800元）
  利润：4000 - 800 = 3200元

═══════════════════════════════════════════════════════════════════
    常见问题解答
═══════════════════════════════════════════════════════════════════

Q1：如何修改系统配置？
A1：编辑文件 "财务数据/配置文件/oxidation_config.json"
    可以修改计价单位、外发工序、费用分类等

Q2：示例数据可以删除吗？
A2：可以。示例数据仅用于学习，不影响实际使用
    位置：财务数据/示例数据/

Q3：如何连接飞书多维表格？
A3：当前版本是演示版，展示核心功能
    连接飞书需要配置 .env 文件中的飞书凭证

Q4：支持哪些银行流水格式？
A4：支持Excel格式的银行流水
    G银行：对公账户，有发票
    N银行/微信：现金账户，无发票

Q5：如何查看详细文档？
A5：查看以下文档：
    - 氧化加工厂版_使用说明.txt（本文件）
    - .kiro/specs/oxidation-factory-optimization/QUICKSTART.md
    - .kiro/specs/oxidation-factory-optimization/README.md

Q6：遇到问题怎么办？
A6：1. 查看使用教程（功能11）
    2. 查看详细文档
    3. 运行测试脚本：python test_standalone_app.py
