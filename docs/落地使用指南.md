# CWZS氧化加工厂财务系统 - 落地使用指南

## 🎯 系统概述

CWZS（氧化加工厂财务系统）是一款专为小型氧化加工厂设计的专业财务管理软件，具有以下特色：

### 🏭 行业特色功能
- **多元化计价支持**：件/条/只/个/米/公斤/平方米七种计价方式
- **委外加工管理**：完整支持喷砂、拉丝、抛光等工序外包
- **灵活财务管理**：G银行(有票)/N银行(现金)双账户管理
- **智能数据处理**：收付款无需一一对应，按实际发生记录

### 🚀 核心优势
- **零技术门槛**：菜单式操作界面，小白也能轻松上手
- **渐进式学习**：从模拟数据到真实业务的平滑过渡
- **原有系统整合**：在CWZS V2.0基础上优化增强
- **一键部署使用**：全自动环境配置，开箱即用

## 📋 快速开始

### 第一步：系统部署

## 🎮 操作流程详解

### 📊 日常工作流程

**早晨开机**：
1. 启动系统查看昨日汇总
2. 检查待处理订单状态
3. 录入新接订单信息

**工作期间**：
1. 记录客户付款（可部分收款）
2. 登记各项支出费用
3. 更新委外加工进度

**下班前**：
1. 查看当日收支汇总
2. 系统自动备份数据

### 🎯 核心功能操作

#### 1. 客户管理
```
功能：添加和管理客户信息
操作：
- 输入客户名称、联系人、电话
- 系统自动生成唯一客户编号
- 支持客户信息查询和修改
```

#### 2. 订单录入
```
功能：记录加工订单信息
支持计价方式：
- 件/条/只/个：标准件加工
- 米：管材、型材加工
- 公斤：原材料加工
- 平方米：板材加工

委外工序：
- 喷砂、拉丝、抛光可选择外包
- 系统自动计算委外费用
- 完整跟踪加工进度
```

#### 3. 收入管理
```
功能：记录客户付款
银行类型：
- G银行：有发票的正规业务
- N银行：现金/微信等

特点：
- 支持部分收款
- 无需与订单一一对应
- 按实际发生时间记录
```

#### 4. 支出管理
```
功能：登记各项费用支出
支出类型：
房租、水电费、三酸、片碱、亚钠、色粉、
除油剂、挂具、外发加工费、日常费用、工资、其他

特点：
- 分类详细，便于成本分析
- 支持供应商信息记录
- 自动生成费用统计报表
```
