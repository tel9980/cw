# 🚀 氧化加工厂财务系统 V2.0 - 1页纸启动指南

## 5分钟上手

### 第1步：启动系统（选一种方式）

**方式A：Web版（推荐）⭐**
```bash
双击: 启动_Web版.bat
打开浏览器: http://localhost:5000
```

**方式B：命令行版**
```bash
双击: 启动_氧化加工厂财务系统_V2.0.bat
```

**方式C：每日工作流**
```bash
双击: 每日工作流.bat
按提示操作
```

### 第2步：熟悉主界面

```
┌─────────────────────────────────────────────────┐
│  氧化加工厂财务系统 V2.0                        │
├─────────────────────────────────────────────────┤

├─────────────────────────────────────────────────┤

└─────────────────────────────────────────────────┘
```

### 第3步：日常操作

| 操作 | 位置 | 说明 |
|------|------|------|
| 新订单 | 首页→[新建订单] | 客户→物品→计价单位→金额 |
| 录入收入 | 首页→[录入收入] | 选择客户→金额→银行类型 |
| 录入支出 | 首页→[录入支出] | 选择类型→金额→备注 |
| 查看报表 | 顶部[报表] | 利润表、客户对账单等 |
| 数据备份 | 工具→备份 | 自动保留30天备份 |

## 核心功能速查

### 📝 订单管理
- **计价单位**: 件、条、只、个、米、公斤、平方米
- **加工工序**: 喷砂→拉丝→抛光→氧化
- **外发加工**: 可选择将前3道工序外包

### 💰 收入/支出
- **收入类型**: 订单收款、预收款、其他收入
- **支出类型**: 房租、水电费、三酸、片碱、亚钠、色粉、除油剂、挂具、外发加工费、日常费用、工资、其他
- **银行账户**: G银行(有票)、N银行(无票/微信)

### 📊 报表
- 利润表（按月/季/年）
- 客户对账单
- 供应商对账单
- 收支明细表

## 常见问题

**Q: 找不到数据？**
A: 检查 `oxidation_finance_v20/` 目录下的 `.db` 文件

**Q: Web版打不开？**
A: 1. 检查Flask是否安装: `pip install flask`
A: 2. 检查端口5000是否被占用

**Q: 需要重新生成测试数据？**
A: 运行: `python oxidation_finance_v20/examples/generate_comprehensive_demo.py`
