## 🌟 系统特色

### 为什么选择这个系统？

✅ **专业** - 完全理解氧化加工厂的业务
- 支持7种计价方式（件、条、只、个、米、公斤、平方米）
- 自动处理委外加工（喷砂、拉丝、抛光）
- 智能管理氧化工序和成本

✅ **简单** - 小白会计也能用
- 内置完整示例数据
- 详细操作说明
- 依葫芦画瓢就能学会

✅ **灵活** - 符合实际业务
- 客户付款不一定对应具体订单？没问题！
- 供应商付款可以灵活分配
- 按实际发生记账

✅ **实用** - 双银行管理
- G银行：有票据的正式交易
- N银行：现金交易（与微信结合）

## 📊 系统功能

### 1. 订单管理

**支持的计价方式**：
- 按件：螺丝、螺母等小零件
- 按条：铝条、钢条等长条形
- 按只：把手、拉手等单个物品
- 按个：配件、组件等
- 按米：铝型材、管材等按长度
- 按公斤：板材、型材等按重量
- 按平方米：铝板、钢板等按面积

**工序管理**：
- 喷砂：去除表面氧化层
- 拉丝：拉出丝纹效果
- 抛光：打磨至光亮
- 氧化：最后工序（工厂完成）

**委外加工**：
- 自动记录委外供应商
- 自动计算委外成本
- 关联到订单利润分析

### 2. 收入管理

**灵活的收款方式**：
- 一对一：一笔收款对应一个订单
- 一对多：一笔收款分配给多个订单
- 预收款：先收款，后分配
- 部分收款：订单分多次收款

**双银行管理**：
- G银行：正式交易，有发票
- N银行：现金交易，与微信结合

### 3. 支出管理

**支出分类**（共12类）：
1. 房租：厂房租金
2. 水电费：公用事业
3. 三酸：硫酸、硝酸、盐酸
4. 片碱：氢氧化钠
5. 亚钠：亚硫酸钠
6. 色粉：氧化着色
7. 除油剂：前处理
8. 挂具：工装夹具
9. 外发加工费：委外费用
10. 日常费用：办公、维修等
11. 工资：员工薪酬
12. 其他：未分类支出

### 4. 财务报表

**自动生成三大报表**：
- 利润表：看赚了多少钱
- 资产负债表：看有多少资产
- 现金流量表：看现金流动情况

**业务分析报告**：
- 客户收入排行
- 产品盈利分析
- 成本结构分析
- 现金流预测
