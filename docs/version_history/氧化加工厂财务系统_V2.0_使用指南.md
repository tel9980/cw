**场景1：客户下了新订单**

**场景3：采购原料**

**场景4：生成财务报表**

## 📊 系统功能详解

### 1. 订单管理

**支持的计价方式**：
- 按件：螺丝、螺母等小零件
- 按条：铝条、钢条等长条形
- 按只：把手、拉手等单个物品
- 按个：配件、组件等
- 按米：铝型材、管材等按长度
- 按公斤：板材、型材等按重量
- 按平方米：铝板、钢板等按面积

**工序管理**：
- 喷砂：去除表面氧化层
- 拉丝：拉出丝纹效果
- 抛光：打磨至光亮
- 氧化：最后工序（工厂完成）

**委外加工**：
- 自动记录委外供应商
- 自动计算委外成本
- 关联到订单利润分析

### 2. 收入管理

**灵活的收款方式**：
```
情况1：客户付款对应一个订单
→ 直接分配给该订单

情况2：客户付款对应多个订单
→ 按比例或金额分配给多个订单

情况3：客户预付款
→ 先记录收入，订单完成后再分配

情况4：客户欠款
→ 系统自动计算应收账款
```

**双银行管理**：
- G银行：正式交易，有发票
- N银行：现金交易，与微信结合

### 3. 支出管理

**支出分类**（共12类）：
1. 房租：厂房租金
2. 水电费：公用事业
3. 三酸：硫酸、硝酸、盐酸
4. 片碱：氢氧化钠
5. 亚钠：亚硫酸钠
6. 色粉：氧化着色
7. 除油剂：前处理
8. 挂具：工装夹具
9. 外发加工费：委外费用
10. 日常费用：办公、维修等
11. 工资：员工薪酬
12. 其他：未分类支出

**供应商管理**：
- 原料供应商
- 委外加工商
- 自动生成对账单

### 4. 财务报表

**自动生成三大报表**：

1. **利润表**
   - 营业收入：加工费收入
   - 营业成本：原料+委外+人工
   - 期间费用：房租+水电+日常
   - 净利润：自动计算

2. **资产负债表**
   - 资产：银行存款+应收账款
   - 负债：应付账款
   - 所有者权益：自动平衡

3. **现金流量表**
   - 经营活动现金流
   - 投资活动现金流
   - 筹资活动现金流

**业务分析报告**：
- 客户收入排行
- 产品盈利分析
- 成本结构分析
- 现金流预测

## 🎓 常见问题解答

### Q1：客户付款不对应订单怎么办？

**答**：没关系！系统支持灵活分配。

```
方法1：先记录收入，以后再分配
1. 记录收入时，不选择订单
2. 以后有订单时，再"分配收入"

方法2：分配给多个订单
1. 记录收入时，点击"分配到多个订单"
2. 选择要分配的订单
3. 输入每个订单的分配金额

方法3：部分分配
1. 先分配一部分给已知订单
2. 剩余部分留待以后分配
```

### Q2：委外加工怎么记账？

**答**：系统自动处理委外加工。

```
记录订单时：
1. 勾选需要委外的工序（喷砂、拉丝、抛光）
2. 填写委外单价
3. 系统自动计算委外成本

支付委外费用时：
1. 记录支出，类型选"外发加工费"
2. 选择委外供应商
3. 关联到对应订单
4. 系统自动更新订单成本
```

### Q3：G银行和N银行有什么区别？

**答**：
- **G银行**：用于有发票的正式交易
  - 客户正式付款
  - 供应商正式采购
  - 需要开票的业务

- **N银行**：用于现金交易
  - 与微信结合使用
  - 小额现金收支
  - 不需要发票的业务

### Q4：如何生成税务申报需要的报表？

**答**：
```
1. 点击"税务报表"
2. 选择申报期间
3. 系统自动生成：
   - 增值税申报表
   - 所得税申报表
   - 财务报表附注
4. 导出Excel
5. 提交给税务局或代理记账公司
```

### Q5：数据会丢失吗？

**答**：不会！系统有三重保护：
1. **自动备份**：每天自动备份数据
2. **手动备份**：随时点击"备份数据"
3. **云端备份**：可选择同步到云端

恢复数据：
```
1. 点击"恢复数据"
2. 选择备份文件
3. 点击"恢复" - 完成！
```
