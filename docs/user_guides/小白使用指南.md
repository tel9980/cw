# 🏭 氧化加工厂财务助手 V2.0 - 小白使用指南

## 🎯 系统特色

专门为氧化加工厂的小会计设计，**零技术基础**也能轻松上手！

### ✅ 完美匹配您的业务需求

**计价方式全覆盖**：
- 🔢 **按件/条/只/个** - 适合标准件加工
- 📏 **按米** - 适合管材、型材加工  
- ⚖️ **按公斤** - 适合重量计价的零部件
- 📐 **按平方米** - 适合板材、大面积加工

**工序管理专业**：
- ⚒️ **委外加工支持** - 喷砂、拉丝、抛光可外包
- 🔄 **完整工艺流程** - 从预处理到最终氧化
- 💰 **成本精确核算** - 自动计算委外费用

**财务管理灵活**：
- 🏦 **双银行账户** - G银行(有票) + N银行(现金/微信)
- 💸 **灵活收付款** - 不需要一一对应，按实际发生记录
- 📊 **自动报表生成** - 利润表、资产负债表一键生成

## 🚀 快速开始三步曲

### 第一步：启动系统
双击 `启动_小白专用.bat` 文件

### 第二步：学习操作（推荐）
选择 `[2] 生成学习用的模拟数据`
- 系统会自动创建6个客户
- 生成10个不同类型订单示例
- 包含完整的收入支出记录

### 第三步：开始使用
选择 `[1] 启动财务管理系统`
按照菜单提示逐步操作

## 📋 日常操作流程

### 📝 每日工作清单

**上午开工**：
1. 启动系统查看昨日汇总
2. 检查待处理订单状态
3. 录入新接订单信息

**工作期间**：
1. 记录客户付款（可部分收款）
2. 登记各项支出费用
3. 更新委外加工进度

**下班前**：
1. 查看当日收支汇总
2. 系统自动备份数据

### 🎯 核心功能操作

#### 1. 添加客户信息
```
选择 [2] 添加客户
输入：客户名称、联系人、电话
系统自动生成客户编号
```

#### 2. 录入加工订单
```
选择 [3] 添加订单
输入：
- 客户ID（从客户列表获取）
- 物品名称
- 数量 + 计价单位（件/米/公斤/平方米等）
- 单价
- 委外工序（如需要）
```

#### 3. 记录客户付款
```
选择 [4] 记录收入
输入：
- 客户ID
- 付款金额
- 银行类型（G银行/N银行）
- 付款说明
```

#### 4. 登记费用支出
```
选择 [5] 记录支出
支出类型包括：
房租、水电费、三酸、片碱、亚钠、色粉、
除油剂、挂具、外发加工费、日常费用、工资、其他
```

## 💡 实用技巧

### 🎯 计价方式选择指南
- **标准件**：选择"件"
- **管材型材**：选择"米" 
- **原材料**：选择"公斤"
- **板材**：选择"平方米"
- **特殊定制**：可根据实际情况选择最接近的单位

### 💰 收付款处理
- **客户付款**：可以一次性付清，也可以分期付款
- **供应商付款**：同样支持部分付款
- **银行区分**：G银行用于有发票的正规业务，N银行用于现金/微信等

### 📊 数据查看
- **实时统计**：随时查看财务概况
- **明细查询**：查看具体的订单、收支明细
- **定期备份**：建议每周备份一次数据文件

## 🆘 常见问题解答

### Q: 不会操作怎么办？
A: 选择生成模拟数据功能，系统会创建完整的示例数据供您学习参考

### Q: 数据安全吗？
A: 所有数据保存在本地JSON文件中，完全由您掌控，建议定期备份

### Q: 可以处理大量数据吗？
A: 当前版本适合中小规模企业，支持数千条记录的管理

### Q: 需要联网吗？
A: 不需要，纯本地运行，保护您的商业信息安全

### Q: 如何生成正式报表？
A: 系统提供财务概况统计，可导出数据配合Excel制作正式报表
