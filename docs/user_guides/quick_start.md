# 快速开始指南

## 🎯 三步上手

### 第一步：启动系统
双击根目录下的 `启动系统.bat` 文件，选择相应功能：

```
[1] 启动 Web界面      - 浏览器访问 http://localhost:5000
[2] 今日概览         - 查看今日统计数据
[3] 智能提醒         - 待加工/未收款提醒
[4] 数据备份         - 备份数据库
[5] 数据恢复         - 从备份恢复
[6] 数据质量检查     - 检查数据完整性
[7] 系统设置向导     - 初始化配置
[8] 每日工作流       - 推荐操作流程
```

### 第二步：使用Web界面
1. 选择 `[1] 启动 Web界面`
2. 浏览器自动打开 `http://localhost:5000`
3. 使用默认账户登录：
   - 用户名：`admin`
   - 密码：`admin123`

### 第三步：开始操作
- 点击左侧菜单进行各项操作
- 系统会自动保存您的数据
- 建议定期进行数据备份

## 📋 每日工作流程

### 早晨（3分钟）
1. 启动Web版系统
2. 查看今日概览和待办事项
3. 检查未收款提醒

### 白天工作
1. 录入新订单
2. 记录收入支出
3. 处理客户往来

### 晚间收尾（2分钟）
1. 查看当日汇总
2. 系统自动备份数据
