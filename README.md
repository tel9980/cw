# 飞书财务小助手 (Feishu Financial Assistant)

全场景自动化财务管理工具，专为小微企业和个人开发者设计。

## ✨ 主要功能

- **🚀 一键全自动**: 自动扫描、导入、对账、体检、备份。
- **🤖 AI 智能分类**: 集成 GLM-4 AI，自动推断交易分类。
- **📂 挂机监听模式**: 自动监听文件夹，Excel 文件一拖即处理。
- **🏦 智能对账**: 自动识别微信/支付宝/银行流水，生成对账单。
- **📊 每日简报**: 终端可视化图表 + 飞书消息推送。
- **🛡️ 财务体检**: 自动扫描重复录入、漏记折旧等风险。

## 📖 使用说明

详细操作指南请查看 [使用手册_小白必读.txt](使用手册_小白必读.txt)

## 🛠️ 快速开始

1. 配置 `.env` 文件 (填入飞书 App ID & Secret)。
2. 运行 `run.bat` 启动程序。
3. 选择 `97` 初始化表格。
4. 开始使用！
