## 系统简介

本系统是专为氧化加工厂定制的财务管理工具，帮助小白会计轻松完成：

✅ **订单管理**：记录加工订单，支持多种计价单位（件/条/只/个/米长/米重/平方）  
✅ **收款对账**：灵活对账，支持多笔合并、分批付款  
✅ **智能分类**：AI自动识别费用类型（原材料、外发加工、固定支出等）  
✅ **报表生成**：一键生成加工费收入、外发成本、利润表等专业报表  
✅ **自动导入**：银行流水拖进文件夹，自动处理

## 典型业务场景

### 场景1：新建加工订单

**业务描述**：华为下单1000件铝合金外壳，单价2.5元/件，需要喷砂和氧化，喷砂外发成本300元。

**操作步骤**：
```
1. 输入 01 (新建加工订单)
2. 按向导填写：
   - 客户名称：华为技术有限公司
   - 订单日期：2026-02-09
   - 物品名称：铝合金外壳
   - 计价单位：件
   - 数量：1000
   - 单价：2.5
   - 工序明细：喷砂、氧化、封孔
   - 外发工序：勾选"喷砂"
   - 外发成本：300
3. 确认创建
```

**系统自动计算**：
- 订单金额：2500元（1000 × 2.5）
- 预计利润：2200元（2500 - 300）

---

### 场景2：客户多笔订单合并付款

**业务描述**：华为有2个订单（PO001: 2500元，PO002: 1800元），一次性转账4300元。

**操作步骤**：
```
1. 先导入银行流水（4300元收款记录）
2. 输入 02 (订单收款对账)
3. 选择客户：华为技术有限公司
4. 在未对账流水中勾选：4300元那笔
5. 在未收款订单中勾选：PO001和PO002
6. 点击"自动匹配"
7. 系统显示：
   - 银行流水：4300元
   - 订单总额：4300元（PO001: 2500 + PO002: 1800）
   - 对账金额：4300元
   - 剩余未收：0元
8. 确认对账
```

**对账后效果**：
- PO001订单：已收款2500元，未收款0元，状态变为"已结算"
- PO002订单：已收款1800元，未收款0元，状态变为"已结算"
- 银行流水：对账状态变为"已对账"

---

### 场景3：客户分批付款

**业务描述**：小米订单3000元，先付1500元预付款，完工后再付1500元尾款。

**操作步骤**：

```

3. 选择客户：小米科技有限公司
4. 勾选1500元流水和对应订单
5. 确认对账
```

**对账后**：
- 订单：已收款1500元，未收款1500元，状态"生产中"

**第二次收款（尾款）**：
```
1. 导入银行流水（1500元收款记录）
2. 重复对账操作
3. 确认对账
```

**对账后**：
- 订单：已收款3000元，未收款0元，状态变为"已结算"

## 常见问题

### Q1：导入银行流水时，系统说"发现重复记录"怎么办？

**A**：系统会自动跳过重复记录，不会重复导入。如果确实需要导入，请检查：
- 日期是否相同
- 金额是否相同
- 业务类型是否相同
- 备注前10个字是否相同

如果以上都相同，系统会认为是重复记录。

---

### Q2：AI分类不准确怎么办？

**A**：
1. 手动修正分类
2. 输入 17 (智能学习)，系统会学习你的修正习惯
3. 下次遇到类似交易，AI会自动使用你修正后的分类

**提示**：修正越多，AI越准确！

---

### Q3：客户名称不一致怎么办？

**例如**：银行流水显示"华为"，但订单里是"华为技术有限公司"

**A**：
1. 输入 12 (管理往来单位)
2. 找到"华为技术有限公司"
3. 在别名列表中添加"华为"
4. 保存
5. 下次导入流水时，系统会自动识别"华为"就是"华为技术有限公司"

---

### Q4：如何批量导入往来单位？

**A**：
1. 打开模板：财务数据\Excel模版\往来单位导入模板.xlsx
2. 按模板格式填写：
   ```
   单位名称          单位类型  别名列表                联系人  联系电话
   华为技术有限公司  客户      华为\n华为科技\nHuawei  张经理  13800138001
   ```
3. 保存文件
4. 输入 12 (管理往来单位)
5. 选择"批量导入"
6. 选择文件
7. 导入完成

---

### Q5：对账错了怎么撤销？

**A**：
1. 输入 02 (订单收款对账)
2. 选择"查看对账历史"
3. 找到错误的对账记录
4. 点击"撤销对账"
5. 系统会自动：
   - 恢复订单的未收款金额
   - 恢复银行流水的对账状态
   - 删除对账记录

---

### Q6：计价单位选错了怎么办？

**例如**：订单应该按"米长"计价，结果选成了"件"

**A**：
1. 输入 20 (加工费收入明细)
2. 找到错误的订单
3. 点击"编辑订单"
4. 修改计价单位和相关数据
5. 保存

**注意**：如果订单已经对账，需要先撤销对账，修改后再重新对账。

---

### Q7：如何设置自动备份？

**A**：
1. 输入 32 (备份与恢复)
2. 选择"设置自动备份"
3. 设置备份时间（建议每天18:00）
4. 系统会自动：
   - 每天定时备份
   - 保留最近30天的备份
   - 备份包含所有配置和数据

---

### Q8：误删数据怎么恢复？

**A**：
1. 输入 32 (备份与恢复)
2. 选择"从备份恢复"
3. 选择备份时间点（例如：昨天18:00）
4. 确认恢复
5. 系统会恢复到选定的时间点

**注意**：恢复后，恢复时间点之后的数据会丢失，请谨慎操作！

---

### Q9：如何导出数据给会计师事务所？

**A**：
1. 输入 24 (月度利润表)
2. 选择时间范围
3. 点击"导出Excel"
4. 文件保存在：财务数据\查询报告

**可导出的报表**：
- 加工费收入明细表
- 外发成本统计表
- 原材料消耗统计表
- 往来对账表
- 月度利润表

---

### Q10：系统运行慢怎么办？

**A**：
1. 输入 30 (系统配置)
2. 选择"清理缓存"
3. 系统会清理：
   - AI分类缓存
   - 报表数据缓存
   - 临时文件
4. 重启系统

**如果还是慢**：
- 检查网络连接（系统需要访问飞书API）
- 检查数据量（超过10000条记录建议归档历史数据）
- 联系技术支持

## 💡 小贴士

### 提示1：善用快捷操作

建议每天下班前：
1. 输入 `32` (备份与恢复)
2. 选择"手动备份"
3. 输入备份说明（例如：2月9日下班前备份）

1. 输入 `02` 完成所有对账
