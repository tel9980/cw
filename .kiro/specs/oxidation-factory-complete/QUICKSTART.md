# V1.7 快速开始指南

## 🎯 5分钟了解V1.7

V1.7是专为**氧化加工厂小白会计**设计的智能财务助手,整合了三个成功版本的精华:

- 💡 **V1.5的智能**: 工作台、工作流、自动化
- 🛠️ **V1.6的实用**: 报表、提醒、对账、导入
- 🏭 **行业特色**: 计价单位、外发加工、行业分类

## ✨ 核心亮点

### 1. 每天打开就知道要做什么
```
智能工作台显示:
✓ 今日待办事项
✓ 超期未收款提醒
✓ 税务申报提醒
✓ 现金流预警
✓ 快捷操作入口
```

### 2. 支持氧化加工厂特殊业务
```
✓ 7种计价单位: 件、条、只、个、米长、米重、平方
✓ 外发加工管理: 喷砂、拉丝、抛光
✓ 行业费用分类: 三酸、片碱、亚钠、色粉、除油剂、挂具
✓ 灵活对账: 一笔对多笔、多笔对一笔
✓ 银行分类: G银行(有票)、N银行/微信(现金)
```

### 3. 一键生成各种报表
```
管理报表: 给老板看的(收支对比、利润趋势、客户排名)
税务报表: 给税务局的(增值税、所得税申报表)
银行报表: 申请贷款用的(资产负债表、利润表、现金流量表)
行业报表: 氧化加工厂专用(加工费明细、外发成本、原材料消耗)
```

### 4. 智能提醒不遗漏
```
✓ 税务申报提醒(提前7天、3天、1天、当天)
✓ 应付账款提醒(到期前提醒)
✓ 应收账款提醒(逾期30天、60天、90天,自动生成催款函)
✓ 现金流预警(预测7天内不足)
```
