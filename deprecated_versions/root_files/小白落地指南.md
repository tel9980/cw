# 🎯 V1.5 小白落地指南 - 3分钟上手

## 📦 第一步：获取代码（3种方式任选1种）

### 方式1：直接下载（最简单）⭐ 推荐
1. 打开浏览器，访问：https://github.com/tel9980/cw
2. 点击绿色按钮 "Code" → "Download ZIP"
3. 下载完成后，解压到任意文件夹（比如：`D:\财务助手`）

### 方式2：使用Git克隆
```bash
git clone https://github.com/tel9980/cw.git
cd cw
```

### 方式3：已有代码直接更新
```bash
git pull origin master
```

## 🔧 第二步：安装Python环境（如果还没有）

### 检查是否已安装Python
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入：`python --version`
3. 如果显示版本号（如 Python 3.9.x），说明已安装，跳到第三步

### 如果没有Python，下载安装：
1. 访问：https://www.python.org/downloads/
2. 下载 Python 3.9 或更高版本
3. 安装时**务必勾选** "Add Python to PATH"
4. 点击 "Install Now"

## ⚡ 第三步：一键安装依赖

### 方法1：双击运行（最简单）⭐ 推荐
直接双击文件：`一键安装.bat`

### 方法2：手动安装
打开命令行（cmd），进入项目文件夹，运行：
```bash
pip install -r requirements.txt
```

## 🆘 遇到问题怎么办？

### 常见问题速查

#### Q1: 双击bat文件没反应？
**解决**：
1. 右键点击bat文件
2. 选择"以管理员身份运行"
3. 或者手动打开cmd，进入文件夹，运行：`python 氧化加工厂财务助手_V1.5实战版.py`

#### Q2: 提示"找不到模块"？
**解决**：
1. 重新运行：`一键安装.bat`
2. 或手动安装：`pip install -r requirements.txt`

#### Q3: 数据保存在哪里？
**回答**：
- 所有数据在 `财务数据` 文件夹
- 自动备份在 `backup` 文件夹
- 每5分钟自动备份一次

#### Q4: 如何备份数据？
**回答**：
- 系统自动备份，不用管
- 手动备份：复制整个 `财务数据` 文件夹
- 建议每周复制一次到U盘或云盘

#### Q5: 可以多台电脑使用吗？
**回答**：
- 可以！复制整个文件夹到另一台电脑
- 或者使用云盘同步 `财务数据` 文件夹
- 注意：不要同时在两台电脑上操作

#### Q6: 手机能用吗？
**回答**：
- V1.5版本支持移动端
- 需要在电脑上运行，手机浏览器访问
- 支持语音输入和拍照识别
