# 🎯 3步开始使用

## 第1步：安装
双击运行：`一键安装.bat`

## 第2步：启动
双击运行：`启动_V1.5实战版.bat`

## 第3步：使用
看到菜单后，选择功能编号，按回车即可

## 🆘 遇到问题？

### 安装失败？
1. 确认已安装Python（3.9+）
2. 重新运行 `一键安装.bat`

### 启动失败？
1. 右键bat文件
2. 选择"以管理员身份运行"

### 不知道怎么用？
1. 先生成测试数据练手
2. 查看 `小白落地指南.md`
3. 查看 `V1.5实战版_使用说明.txt`
